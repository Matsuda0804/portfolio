SKILL

基本的なスキルから応用まで幅広いWEB制作スキルを扱えます。
お客様の要望に合わせ、きちんと効果のあるサイト制作を行うことが可能です。

HTML5

SEOを意識し、更新や改修作業をしやすいシンプルなコーディングを行います。構文チェックなどを行い、HTML5で今の時代に合う最先端の構築が可能です。

CSS3

モバイルファーストを意識したレスポンシブ（スマホ）対応はもちろん、CSSでの簡単なアニメーションや動的な装飾を行う時ことで、UIを意識したデザインを行います。

JavaScript / jQuery

現代のサイト制作によくある動作（スクロール、ドロップダウン、タブ切り替え、スライド、パララックスなど）を使用しあらゆる動的なサイト制作が可能です。

PHP

簡単なフォーム作成やその他基礎知識および、Wordpressカスタマイズに必要なスキルがございます。

WordPress

静的サイトのWordPress化、テーマのカスタマイズやオリジナルテーマの作成が行えます。更新頻度の高いサイトにオススメです。

Photoshop

MVやKV、バナーや広告サムネイルの制作、簡単なグラフィック技術を使用しデザインに凝った画像の編集が可能です。最近はYouTubeのサムネイル制作なども行なっています。

Illustrator

サイトに使用する簡単な素材、ワンポイント画像、要望に沿った簡易的な企業ロゴなど制作致します。DTPオペレーターとしての経験もあるため名刺作成などの簡単な紙媒体デザインも可能です。

XD

デザインの基となるワイヤーフレームやデザインカンプ、プロトタイプの制作に使用します。シンプルなデザインのサイトやLPであればXDを使用してより効率的にサイトデザインを制作することが可能です。
